About

My name is Rémy, I am a Web Lover and I focus myself on Front-end Development and Webdesign.

I started coding after some deep thinkings when I got 21 and my Marketing bachelor. Those concerns where about creativity, motivation and learning which I have been able to fully rediscover when I entered the web. I followed a huge learning hunger which kept myself motivated through a long and sometimes complicated journey. As a self-taught, I must thank various brilliant and essential platforms :
OpenClassrooms, FreeCodecamp and CodeCademy.

My self-teaching journey coming to its primary goal of mastering main front-end languages and technologies, I am now looking for professional challenges. I code everyday as a hobby and for a living with the powerful trio HTML - CSS - JS, but I also use powerful tools such as Bootstrap and jQuery, and some software fundamentals which are Photoshop and Illustrator.

I invite you to check different projects I realised for a better overview of my skills in the Creations section.
If you have any question, comment or want to see my resume, feel free to reach the Contact section.
